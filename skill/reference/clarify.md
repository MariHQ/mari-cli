# clarify — rewrite unclear or confusing copy

Make the meaning unmistakable.

## Flow
1. Run the detector; read clarity + inclusive findings.
2. Apply:
   - define jargon and expand acronyms on first use (`undefined-acronym`)
   - convert passive → active where the actor matters (`passive-voice`)
   - replace nominalizations with verbs (`nominalization`: "make a decision" → "decide")
   - simplify complex words (`complex-word`)
   - resolve ambiguous pronouns and dangling references
   - fix the error-message formula: **what happened / why / how to fix**
   - replace vague link text with a description (`vague-link-text`)
3. Keep it true to the source; clarifying is not rewriting the argument.

Leans on: `passive-voice`, `undefined-acronym`, `complex-word`, `nominalization`,
`there-is-expletive`, `vague-link-text`.
